# Voxel Engine

A high-performance C++ voxel engine with modern rendering capabilities.

## Current Iteration: 4

## Features

- Chunk-based voxel rendering
- OpenGL-based graphics
- Lua scripting support
- Modular architecture
- Physics simulation
- Audio support
- Networking capabilities

## Building

```bash
mkdir build && cd build
cmake ..
make -j$(nproc)
```

## Running

```bash
./voxel_engine_app
```

## Development

This project is automatically built and updated by NEXUS autonomous runner.
Each iteration adds new features and improvements.
